### Techstack

- React
- React Router DOM
- React-query
- Zustand
- Material UI
- React Hook Form
- Yup

### Prerequisite

- Nodejs (Latest LTS version)
- pnpm (version 8.6.12) - To install specific version run

```sh
npm i -g pnpm@8.6.12
```

### 1. Install dependencies

```sh
pnpm install
```

### 2. Start Development

```sh
pnpm dev
```
